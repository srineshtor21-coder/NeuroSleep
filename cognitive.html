📚 What Each Test Measures

⚡ Reaction Time

Measures neural processing speed via the prefrontal cortex and anterior cingulate. Normal range: 150-280ms. Sleep deprivation adds 40-60ms on average. Variability across trials is often more diagnostic than mean alone.

🧠 Working Memory

Spatial sequence task based on the Corsi Block Test. Taps visuospatial working memory capacity, typically 5-9 items. Sleep loss impairs prefrontal-hippocampal coordination, reducing both capacity and accuracy.

🔮 Pattern Recognition

Tests executive function and visual reasoning via sequence completion. Requires dorsal prefrontal and parietal cortex. Performance declines measurably after poor sleep, reflecting disrupted thalamo-cortical neural gating.

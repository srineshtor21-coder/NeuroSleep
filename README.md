# NeuroSleep

**A Digital Platform for Analyzing Sleep Patterns, Cognitive Performance, and Long-Term Alzheimer's Risk**

Created by **Mithil Gajula** : ISM Research Project

---

## About

NeuroSleep is a research-driven digital platform exploring the relationship between sleep consistency, cognitive performance, and Alzheimer's disease risk. It combines behavioral tracking, neuropsychological testing, and data visualization to model how everyday sleep habits may influence long-term brain health.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — overview, science, features |
| `pages/research.html` | Research library with peer-reviewed citations |
| `pages/tracker.html` | Sleep logging and weekly analysis |
| `pages/cognitive.html` | Reaction time, memory, and pattern tests |
| `pages/dashboard.html` | Correlation graphs and data analytics |
| `pages/risk.html` | Alzheimer's risk behavior calculator |

## Deployment (GitHub Pages)

1. Push this repository to GitHub
2. Go to **Settings > Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/neurosleep/`

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js 4.4 (via CDN)
- Google Fonts: Syne + DM Sans
- localStorage for data persistence
